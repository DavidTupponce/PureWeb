Le serveur

Le serveur PureWeb est disponible gratuitement pour Windows et Linux (Testé sous Ubuntu 16.04).
L'utilisation du serveur PureWeb est totalement libre dans n'importe quel contexte, cependant le créateur de ce projet ne saurait être tenu pour responsable d'une quelconque mauvaise utilisation du serveur ayant pour résultat d'enfreindre la loi ou d'apporter des dommages à quelque systéme ou personne que ce soit. L'utilisateur est seul responsable de l'usage qu'il fait du serveur et des fichiers qu'il met en ligne à l'aide de ce serveur. Il est conscient du risque sécuritaire potentiel.
Sélectionnez la version adaptée à votre systéme d'exploitation ci dessous.

Windows (x64) Linux

Versions :

  • Windows : PureWeb 1.4.2 - 15/02/2017
  • Linux (Ubuntu) : PureWeb 1.4 - 08/02/2017

Le code source

En tant que projet libre, les sources du serveur PureWeb sont disponibles également en téléchargement.
Pour la compilation, la possession d'une version de PureBasic (même la démo gratuite) est nécessaire. Il est recommandé d'utiliser la version la plus récente disponible.
